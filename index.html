Dúvidas

O que a conta KidCash oferece?

A conta KidCash oferece uma plataforma segura para gerenciar finanças, ensino sobre economia, ferramentas para poupança e transações simplificadas.

Como faço para abrir uma conta?

Para abrir uma conta KidCash, basta acessar o site, preencher o formulário de inscrição e enviar os documentos solicitados.

Quais documentos são necessários para abrir uma conta?

Para abrir uma conta, você precisa de um documento seu, como a certidão de nascimento, e um documento do seu responsável.

Qual a idade para abrir conta?

De 0 a 12 anos :)
